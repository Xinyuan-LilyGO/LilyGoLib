### Extension interface

```bash
|---------------------------------------------------|
|     | SCL | SDA | MISO  | SCK  | TX | GND  |      |
|     | 5V  | CE  | GPIO9 | MOSI | RX | 3.3V |      |
|---------------------------------------------------|

* CE is XL9555 GPIO9
* TX is ESP32-S3 GPIO43
* RX is ESP32-S3 GPIO44
* MISO is ESP32-S3 GPIO33
* MOSI is ESP32-S3 GPIO34
* SCK is ESP32-S3 GPIO35
* SDA is ESP32-S3 GPIO3
* SCL is ESP32-S3 GPIO2

```
